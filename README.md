# CobraDose: Copula-based Bayesian model averaging for dose selection

This package implements a framework for posterior predictive inference based on two endpoints that models dependence via copulas and accounts for uncertainty in both marginal distributions and dependence structures through Bayesian model averaging

You can install this version from Github:

```
install.packages('devtools')
devtools::install_github("UQ-ULTRA/CobraDose", ref="main")
```
